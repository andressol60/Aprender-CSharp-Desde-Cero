#  Punto de Venta Web
Sistema de facturación básico con arquitectura limpia.

## ️ Stack
- **Backend:** .NET 8, C#, EF Core, SQL Server
- **Frontend:** Vanilla JS, HTML5, CSS3
- **Herramientas:** Git, GitHub, VS Code, SSMS

## 🚀 Ejecución Local
1. Backend: `dotnet run` en `Backend/PuntoDeVenta.API`
2. Frontend: Live Server en `Frontend/index.html`
3. BD: Restaurar `FacturacionDB.bak` o ejecutar `scripts/init.sql`

## 🔐 Configuración
Edita `appsettings.json` con tu cadena de conexión local.
